## Cheatsheet

#### Zip a file

```
zip -r zip_name.zip target_folder
```

#### Copy contents of a file

```
cat filename | xclip -selection clipboard
```

#### Remove all zone identifiers on wsl

```
find . -name "*:Zone.Identifier" -type f -delete
```

#### Fix incorrect windows time on linux dual boot

```
timedatectl set-local-rtc 1 --adjust-system-clock
```

#### Change file permission of a mounted drive

```
sudo chmod u=rwx,g=rwx,o= /dev/mounted-drive
```

#### Check all file sizes of the system excluding /media and /mnt

```
ncdu --exclude /media --exclude /mnt /
```
